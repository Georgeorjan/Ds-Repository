# QueryMzuni Platform

Welcome to **QueryMzuni** — a data science students' platform designed to empower learning and community connection at Mzuzu University.

##  Overview

QueryMzuni provides curated learning resources and a collaborative community for data science enthusiasts. Access the latest materials, connect with peers, and enhance your skills in a supportive environment.

##  Features

- **Responsive Design:** Sleek, modern look with mobile-friendly navigation.
- **Curated Resources:** Python basics, machine learning, and statistics reading materials.
- **Active Community:** Discussion forums to ask questions and share knowledge.
- **Accessible UI:** Easy to use and navigate for all students.

##  Getting Started

1. **Clone the Repository**
    ```sh
    git clone https://github.com/Georgeorjan/Ds-Repository
    cd querymzuni
    ```

2. **Open `index.html` in your browser**

No build step required—just make sure the following files are in the same directory:
- `index.html`
- `style.css`
- `script.js`

##  File Structure

```
.
├── index.html        # Main webpage
├── style.css         # Stylesheet for the site
├── script.js         # JavaScript for interactivity (mobile menu, etc.)
└── README.md         # Documentation (this file)
```
